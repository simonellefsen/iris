# Runbooks

Repeatable operational procedures for Iris.

- [build-test-deploy.md](build-test-deploy.md) — local dev, tests, production build, PWA verify,
  Vercel deploy.
- [manual-qa.md](manual-qa.md) — the on-device acceptance walkthrough (desktop Chrome + iOS Safari).

Iris has no backend and no servers to operate, so runbooks are about the build, the PWA, and
on-device behaviour — not infrastructure.
